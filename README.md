# Rusty S3 Active Storage

S3 Active Storage server

## Pre-commit hook

A pre-commit hook is provided in `tools/pre-commit` that runs formatting,
clippy, and unit tests. After cloning this repository, copy it to
`.git/hooks/pre-commit`.

## Build

```bash
make build
```

## Test

```bash
make test
```

## Run

```bash
make run
```
